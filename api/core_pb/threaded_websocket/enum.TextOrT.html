Enum TextOrT

source 
pub enum TextOrT<T: Debug> {
    Text(String),
    T(T),
    Bytes(Vec<u8>),
}
Expand description

Represents data that is either the given type, or text

Used for websockets; the type is transferred using serialization in bytes, text is sent as regular text
